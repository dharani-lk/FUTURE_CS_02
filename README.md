# Phishing Email Detection & Awareness System

## Project Description
This repository contains my submission for **Cyber Security Task 2 (2026)** under the **Future Interns Internship Program**.

The project focuses on analyzing phishing email samples, identifying phishing indicators, classifying email risks, and creating awareness guidelines to help users avoid phishing attacks.

## Contents of the Repository
- Phishing_Detection_Awareness_Report.pdf – Final awareness report
- Samples/ – Folder containing phishing and legitimate email samples

## Tools Used
- Public phishing email datasets (study and learning purpose only)
- Google Message Header Analyzer
- MXToolbox Email Header Analyzer
- Browser-based URL inspection
- MS Word / PDF documentation

## Key Learnings
- Identifying phishing indicators in emails
- Understanding email-based social engineering attacks
- Classifying emails as Safe or Phishing
- Creating security awareness documentation
